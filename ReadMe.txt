This folder provides R scripts and datafor Sudo(2025) published in Social Science Japan Journal.

00_Vaccine.R: make data for the vaccination rate at the prefecture level
01_wide_to_long_2.R: change wide format to long format (SSPW2021)
02_rate.R: make data for Covid19 at the prefecture level
03_analysis.R: make data for the analyses
04_figure1_2.R: make Figures 1 and 2
05_figure4.R: make Figure 4
06_table1.R: make Table 1
07_t2_f5_ta1.R: make Table 2 and Figure 5
08_t3_f6_ta2.R: make Tabke 3 and Figure 6
09_f7.R: make Figure 7
10_ap1.R: make Table A1
11_ap2_ta3.R: make Table A2
12_a3.R: make Table A3

Covid19_d.csv: number of death caused by Covid19
Covid19_p.csv: number of newly infections 
Covid19_s.csv: number of severe cases
population.csv: population 
prefecture.ndjson: number of vaccinated
